Self Project-AWS CloudFormation, VPC


Deployed Infrastructure as Code using AWS CloudFormation with YAML configuration file, created VPC system with AWS CLI command; Used Lucidchart to implement cloud web server application architecture diagram, created a Multi-Availability Zone Cloud container with Virtual Private Cloud(VPC), added public/private subnets and internet gateway to control traffic;

Implemented basic Networking Infrastructure using CloudFormation code in YAML, added Routing method at the end to configure network routing rules. You can find this part of the code at Github CloudDevND_Networking_Infra page.

Applied Autoscaling group to manage multiple instances, contributed traffic with Load Balancer, especially controlled inbound traffic with Security Groups, used LaunchConfiguration to implement Web Server Application. You can find all the source code in my Github CloudDevND_Server page.
