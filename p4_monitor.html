Self Project-C++, System Monitor

Used C++ created an application which actively monitor running system processes and usage of system resources;
Applied data extraction method to get process information from Linux based Operating System in order to actively read and display information on the live API, used "ncurses" C++ library to create terminal for data display;
Implemented Advanced Objective Oriented Programming in C++ to create multiple composited and inherired classes to hold different type of data and process different tasks as well as classes related by generic programming;
Completed read data from Linux file system, structure the data, process it and then format it to output to terminal;

You can find all the source code in my Github CplusND_System_Monitor page.
