Self Project-C++, System Monitor


Used C++ to implement an application which actively monitors running system processes and usage of system resources;

Applied data extraction method to get process information from Linux based Operating System, actively read and display information on the live API, used "ncurses" C++ library to create a terminal GUI for data display;

Implemented Advanced Object-Oriented Programming in C++ to create multiple composited and inherited C++ classes to hold different types of data, process different tasks, create classes related by generic programming;

Completed read data from the Linux file system, structured the data, processed all the useful information and formatted data to output to System-Monitor terminal GUI;

You can find all the source code in my Github CplusND_System_Monitor page.
