Self Project-C++, Garbage Collector


Implemented memory management with C++ to build garbage collection resource, which cleans up the used memory of a program;

Used stack memory to manage local variables inside the function to complete the fast transaction, used heap memory to manage dynamic variables which can be resized during implementation;

Applied memory relocation and deallocation while managing heap memory to ensure the memory is properly freed after usage;
Worked with pointers and references, and also smart pointers to implement modern C++ programming. Used memory deallocation on all the created heap memory to prevent memory leakage;

You can find all the source code in my Github CplusND-Garbage-Collector.
